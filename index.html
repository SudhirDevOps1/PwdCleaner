Advanced Tools

⚡ One-Click Auto Clean & Merge

Imported from multiple browsers/managers? Re-imported your vault? This finds all exact duplicates (same website + username + password) and removes extras in one click — keeping the best entry from each group.

🔐 Encrypted Vault File (.pwdcleaner)

Download your entire vault as one AES-256-GCM encrypted file. Zero-knowledge — only your master passphrase can decrypt it. Share across devices safely.

Passphrase: same as lock screen

🧹 Auto-Clean After Export

Automatically clear all imported data from browser after exporting — so reimporting the same file won't show old duplicates.

Backup & Restore

Create plain JSON backup of all data or restore from a backup file.

Smart Merge Entries

Automatically merge entries that share the same website and username, combining notes and keeping the best data.

Clean Empty Entries

Remove entries that have no password, no username, or are completely empty.

Normalize URLs

Standardize all URLs — remove trailing slashes, www prefix, and ensure consistent format.

Export Selected Entries

Export only the entries you've selected in the search/filter section above.
